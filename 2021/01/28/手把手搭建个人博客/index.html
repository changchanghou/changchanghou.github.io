Github + Hexo搭建个人博客

现在有很多网站可以用来写博客，但是博客网站的主题各式各样，甚至有的页面繁杂，不能专注于博客阅读。如果你有这样的烦恼，就开始选择自己喜欢的主题搭建自己的博客吧~

1. 环境准备

本文所使用的环境：

  • windows10
  • git@2.28.0
  • node.js@v12.18.3
  • hexo@4.2.0

另外，需要注册github账号。

2. 搭建github博客

新建一个名为你的github用户名.github.io仓库。比如github的用户名是zhangsan，那就建一个zhangsan.github.io仓库，以后博客访问地址就是http://zhangsan.github.io。由此可见每个github账户最多只能创建一个这样的仓库。
